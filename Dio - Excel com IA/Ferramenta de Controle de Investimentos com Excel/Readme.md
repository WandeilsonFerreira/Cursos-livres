# DIO – Ferramenta de Controle de Investimentos com Excel

## Introdução  
A presente atividade tem como objetivo desenvolver uma planilha inteligente no Excel para auxiliar no **controle e análise de investimentos**, permitindo ao usuário simular cenários e acompanhar a evolução do patrimônio de forma prática e visual.

## Perguntas de Negócio   
- Qual é o valor atual do patrimônio?  
- Como diferentes tipos de fundos impactam o resultado?  
- Quais cenários de investimento podem gerar maior retorno?  

## Pré-requisitos  
- Conhecimentos básicos de Excel (fórmulas, formatação, tabelas).  
- Noções de investimentos e tipos de fundos.  
- Organização de dados em planilhas.  

## Base da Tabela  
Criação de uma tabela estruturada com informações essenciais:  
- Tipos de fundos.  
- Valores aplicados.  
- Rentabilidade esperada.  
- Datas de aplicação e resgate.  

## Simulador de Patrimônio  
Ferramenta dentro da planilha que calcula automaticamente o **patrimônio acumulado** com base nos investimentos registrados.

## Simulador de Cenários  
Permite testar diferentes hipóteses de rentabilidade e aportes, ajudando a visualizar **possíveis resultados futuros**.

## Conclusão  
A atividade proporciona ao aluno a prática de **modelagem financeira no Excel**, unindo conceitos de investimentos com técnicas avançadas de planilhas. O resultado é uma ferramenta funcional para tomada de decisão.
